  • connect high-resolution (second) screen
  • make a command set with xrandr of high-resolution configuration, like 2k. There are 3 commands we can put them in *.sh file, the 3rd line is to use xrandr to set primary monitor and turn off the laptop‚Äôs screen.
  • finally, just find a way to execute the 3 commands, you can run them with ‚Äòsh *.sh‚Äô or let them executed automatically after we login system.

Note: the original system doesn‚Äôt support higher resolution more than 1080, so we need to add new resolution and trigger the change.

X.2 when desktop crash

like stucking when turning off some windows or stucking after running something for a long time. When this happened, do NOT shut down the system by cut down the power! that‚Äôs a dangerous behavior, system core may be ruined by this.

the right way is: Ctrl+Alt+F2/3/4, log into the tty2/3/4, then restart gdm/lightdm, or rollback the wrong options here if you remember. Or do nothing, just wait for the system rerun again, sometimes it could be reworked after waiting a while.
